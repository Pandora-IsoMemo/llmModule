# Has Internet

Has Internet

## Usage

``` r
has_internet(url = "https://www.google.com", timeout = 2)
```

## Arguments

- url:

  (character) URL to test for internet connectivity. Default is
  "https://www.google.com".

- timeout:

  (numeric) number of seconds to wait for a response until giving up.
  Can not be less than 1 ms.
